## 2. Herramientas y Librerías de Software
Procesamiento de datos, análisis de lenguaje natural (NLP) y la exportación de los elementos visuales adjuntos, se utilizaron las siguientes herramientas:

* **spaCy / NLTK (Natural Language Toolkit):** Utilizados para la tokenización y el etiquetado de categorías gramaticales.
* **Pandas & NumPy:** Herramientas para la limpieza de datos, estructuración de dataframes y el cálculo del promedio métricas.
* **Matplotlib / Seaborn / Plotly:** Librerias para los gráfico, utilizados para la creación de las visualizaciones y la comparativas.
